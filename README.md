# Personal-Portfolio

This portfolio is made with React and lots of love. Although it is my personal portfolio, feel free to fork and have fun with it.

## Installation

1. Run `npm install`.
2. Run `npm run start`.
3. Visit [localhost:3000](http://localhost:3000).
